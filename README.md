# Medical Records Request Tracker

A beginner-friendly web application that tracks medical records requests, request status, dates, and notes.

## Purpose

This project demonstrates a simple healthcare operations workflow. It helps organize medical records requests by patient name, request type, request date, status, and notes.

## Features

- Add medical records requests
- Track request type
- Track request date
- Track request status
- Add completion or follow-up notes
- View requests in a table
- Beginner-friendly HTML, CSS, and JavaScript project

## Example Use Case

A healthcare support representative can enter a request, track whether it is pending, in progress, completed, or escalated, and document important notes.

## Important Note

This project is for educational and portfolio purposes only. Do not enter real patient information, protected health information, or confidential medical data.
